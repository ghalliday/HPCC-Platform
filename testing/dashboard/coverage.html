Test Purpose

Coverage tests measure the percentage of code that is exercised by the test suites. This metric helps identify untested code paths and ensures comprehensive test coverage across the HPCC Platform codebase.

The coverage analysis tracks which lines of code are executed during test runs, helping developers identify areas that need additional testing and ensuring that critical code paths are adequately validated.
