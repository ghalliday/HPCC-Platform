Test Purpose

Build Verification Tests (BVT) are a focused set of smoke tests that verify the basic functionality of a build before more extensive testing begins. These tests are designed to quickly identify major issues that would prevent further testing.

The BVT suite contains approximately 525 critical test cases that must pass for a build to be considered viable for further testing and deployment.
