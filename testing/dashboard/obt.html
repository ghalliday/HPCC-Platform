Test Purpose

The Overnight Build Test (OBT) is an automated test suite that runs on each nightly build to catch integration issues early. It focuses on critical functionality and common use cases that must work reliably across all supported platforms.

This suite contains approximately 900 test cases covering essential features, data operations, and system integrations that are critical for daily operations.
