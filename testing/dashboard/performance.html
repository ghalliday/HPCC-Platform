Test Purpose

Performance tests measure the execution speed and resource utilization of the HPCC Platform across different workloads. These tests help identify performance regressions and validate that optimizations are effective.

The performance suite runs standardized benchmark queries and compares execution times against established baselines. Results are shown as percentage differences from baseline performance, where negative values indicate improvements (faster execution).
