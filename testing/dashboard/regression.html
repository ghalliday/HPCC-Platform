Test Purpose

Regression tests ensure that new code changes do not break existing functionality. This comprehensive test suite runs across all platforms (hthor, Thor, and Roxie) and versions to validate that previously working features continue to function correctly after updates.

The regression test suite contains over 1,250 individual test cases covering core ECL language features, platform capabilities, and common usage patterns.
